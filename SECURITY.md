# Security

Please report security issues privately through GitHub's security advisory flow
for this repository. Do not include secrets or sensitive local paths in a public
issue.

Bookshelf runs locally and makes no network requests during normal browsing,
quote selection, or ambient delivery. Agent plugins and extensions execute with
the user's permissions. Review their source and host trust prompts before
enabling them.

Supported security fixes target the latest release. Include the Bookshelf
version, operating system, host agent, reproduction steps, and the smallest
useful proof of impact.
