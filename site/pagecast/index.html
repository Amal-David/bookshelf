Useful waiting, by design

A small literary reset between agent turns.

Bookshelf counts completed turns locally and stays quiet until its cadence is due. Then it surfaces one compact quote through the same Stop-hook path packaged for Codex and Claude Code.

$ bookshelf quote --intent refactor

“Do nothing which is of no use.” — Miyamoto Musashi, The Book of Five Rings

$ bookshelf feedback up|down
  • Intent → tags, locally. For example, refactor maps to simplicity, discipline, and focus.
  • Feedback stays local. bookshelf feedback up|down adjusts a future local pick; it does not report a task back to a service.

Bounded by design

Enough context to be useful. Nothing else.

Bookshelf accepts an explicit intent or a bounded host metadata label only. It does not accept or inspect commands, paths, prompts, transcripts, code, tool arguments, model output, or network calls.

  • Ambient delivery is optional and off by default.
  • If the host cannot provide the bounded signal safely, matching fails closed.
  • Adapters keep their own errors contained; this is not a guarantee about a host turn.

Where it meets you

Inside the coding session. Not in another tab.
